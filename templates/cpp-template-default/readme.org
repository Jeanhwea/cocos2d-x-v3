* 快速开始
** Windows 开发
生成项目解决方案文件 *.sln
#+BEGIN_SRC sh
  cmake -B build -G"Visual Studio 17 2022" -A win32
#+END_SRC

构建项目
#+BEGIN_SRC sh
  cmake --build build
#+END_SRC

** MacOS 开发
#+BEGIN_SRC sh
  cmake -B build -G"Xcode"
#+END_SRC

构建项目
#+BEGIN_SRC sh
  cmake --build build --config Release
#+END_SRC
